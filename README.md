# BezoutNoModulo
Bezout looking for no modulo

Permet d'obtenir u et v dans une "ue1*ve2=1" sans passer par l'utilisation explicite de modulo.
Utile pour l'utilisation de bezout ou pour tout simplement trouver

Utilisation:
De préférence, avec l'utilité première que l'on connaît de la formule bezout, e1 et e2 sont deux nombres premiers.
Restriction: e1 > e2 (Peut être modifier dans le code pour faciliter la démarche, toutefois: "aka to do")
